Хочу создать проект графа знаний по Москве, куда бы входила разнообразная информация по городу: история, дороги, адреса, заведения и организации и прочее.
Данные будут лежать в графовой БД Neo4j. 
Пользователь будет обращаться к БД двумя способами:
1) Написав cypher-запрос для извлечения нужных данных
2) Спросив асситента на ествественном языке, который будучи AI агентом сгенерирует нужный cypher-запрос и обратиться к БД, как к инструменту.

План разработки проекта:
1. Начни с проктирования онтологии графа: определить сущности, связи и атрибуты.
2. На втором этапе нужно написать скрипты для сбора данных по открытым источникам и подготовки данных для загрузки в БД Neo4j. 
3. Настроить инфраструктуру Neo4j в docker с учетом apoc драйвера.
4. Написать код AI агента для генерации Cypher-запросов по запросу от пользователя на естественном языке. Написать код подключения AI агента к БД Neo4j, как к инструменту.
5. Написать backend на FastAPI c основными методами: загрузка данных в БД, написание Cypher-запроса к БД, написание запроса к AI агенту.
